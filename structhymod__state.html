Detailed Description

Hymod state structure.

This structure provides storage for the state used by hymod hydrological model at a particular time step. Warning: to allow bindings to other languages the storage amounts for the reserviors in the nash cascade are stored in a pointer which is not allocated or deallocated by this code. Backing storage for these arrays must be provide and managed by the creator of these structures.
